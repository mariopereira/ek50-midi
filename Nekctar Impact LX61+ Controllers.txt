Nektar Impact LX61+ Controllers

General info
------------
- Midi channels are counted from 0
- Modes (buttons below the display)
  * Preset - Sends in the configured MIDI channel
  * Mixer and Inst - Sends in the global MIDI channel
- Message types
  * Toggle - Controller sends a message of value 0 or 127, alternating
  * DoubleMsg - Controller sends a message of value 127 followed by one of value 0
- Values with an * means that there is a default function for that control message
